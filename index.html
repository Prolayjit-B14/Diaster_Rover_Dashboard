Platform Capabilities

End-to-End Disaster Intelligence

Unified monitoring stack from sensor telemetry to field response coordination.

Live Telemetry
Real-time sensor data streamed via MQTT from ESP32. Temperature, gas, vibration, humidity updated at 100ms intervals.
GPS Tracking
Live rover location with NEO-6M GPS module. Interactive Leaflet map with path trail, waypoints and hazard zone overlays.
Hazard Detection
MQ-2 gas sensor, flame detector and PIR motion sensor detect fire, toxic gases and human presence in real-time.
Vision Analysis
ESP32-CAM live streaming with AI object detection, human presence analysis, motion tracking and snapshot capture.
Emergency Alerts
Priority-classified alert system with critical / high / medium / low tiers. SOS broadcast, audio alerts and timeline logging.
Rover Control
Emergency stop, autonomous mode toggle, camera control and return-to-base commands dispatched via MQTT.
